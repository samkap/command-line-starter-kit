THINGS AND STUF

The Command Line Starter Kit

General Command Line commands

~

Go to local root folder.

cd foldername

Change directory into foldername.

..

Go up/out one directory.

ls

lists items inside current folder

&&

Put two commands in one line and separate it with &&.

pwd

prints working directory

open
foldername

Will open finder window for foldername.

mkdir foldername

Create a directory foldername inside current directory.

touch filename.ext

Creates a file (remember to include the file extension).

say

Will read aloud anything stated after say command.

Git commands

git status

Checking the status of new, modified or deleted files.

git add .

Adds changes to the stage.

git commit -m

Written as git commit -m "commit message goes here"

git push origin master

push changes to remote repo

git init

initialize git locally.

git remote add repo-url

Connects remote repo to local repo.

git remote -v

print name and URL on remote repo.

rm -rf .git

USE SPARINGLY and TYPE CAREFULLY

remove a .git file / repo

git checkout -b branchname

creates a new branch

git checkout branchname

Switches to a branch

git checkout -

Switches back to last branch

git pull branchname

Pulls changes from another branch

git clone repo-url

clone a remote repo, locally

git merge branchname

Updates current branch with contents of other branchname

git fetch

import commits from Github/remote into local.

git stash

Takes your modified files and staged changes and saves them so that you can reapply at any time

git stash pop

Reapply the most recent changes that you stashed

git diff

Visually shows the changes you have made since you last staged the file for commit. (Quit with Q.)
